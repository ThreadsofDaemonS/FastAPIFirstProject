# FastAPIFirstProject

## Description
Проект включает:
- **Backend**: FastAPI
- **Frontend**: Vue.js
- **Database**: PostgreSQL

---

## Requirements

- Docker
- Docker Compose

---

## How to start the project

1. Clone repository:
   ```bash
   git clone <repository_url>
   cd FastAPIFirstProject
    ```

2. Start docker-compose
``` 
docker-compose up -d
```
3. If you chacnge something in the project:
```commandline
docker-compose up -d --build
```
